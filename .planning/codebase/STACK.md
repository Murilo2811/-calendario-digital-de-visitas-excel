# Tech Stack

## Core Technologies
- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript

## Styling
- **CSS Framework**: Tailwind CSS
- **Utilities**: tailwind-merge, clsx
- **Icons**: lucide-react

## Data & Persistence
- **Database Replacement**: Excel (xlsx library)
- **File Access**: File System Access API (Browser-native)
- **Local Storage**: IndexedDB (for file handle persistence)

## Utilities
- **Date Handling**: date-fns
